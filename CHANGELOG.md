# Changelog

## 1.0.0 - 2026-01-19

- Initial public release of the Expert Context Pack (ECP) v1.0 specification and reference implementation.
- Published to PyPI as `ecp-reference`: https://pypi.org/project/ecp-reference/1.0.0/
- Includes the `ecpctl` CLI, normative JSON Schemas, evaluation harness, packaging verification, and example packs.
- Adds `CONTRIBUTING.md` and an examples index (`examples/README.md`) for easier community onboarding.
- Expands test coverage (vector payload edge cases, malformed YAML handling, end-to-end conformance pack demo flow).
